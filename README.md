После того, как вы создали свой сайт и подписали договор с ePay,
остается добавить модуль EPAY2 Halyk Bank в Tilda.

Для подключения модуля следуйте инструкции или посмотрите видеоинструкцию:
1.	Авторизуйтесь на Tilda.cc в раздел «Мои сайты»
2.	Выберите свой интернет-магазин (Редактировать сайт)
3.	Выберите «Настройка сайта»
4.	В списке выберите «EPAY2 Halyk Bank» и заполните все поля
В процессе регистрации, команда ePay предоставляет данные. Некоторые из них нужно будет сопоставить в форму tilda:
ClientID в форму «Логин»,
TildaSecret в форму «Секрет для подписи заказа»,
TerminalID в форму «TERMINALID»
5.	Готово! Теперь Вы используете интернет-эквайринг.
